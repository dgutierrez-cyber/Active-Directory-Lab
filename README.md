1. Overview

Built a virtual Active Directory lab to simulate real-world attack scenarios.

2. Environment
Windows Server 2022 (Domain Controller) - ip address = 192.168.56.10
Kali Linux (Attacker) - ip address = 192.168.56.30

4. Objectives
Simulate weak credential attack
Perform password spraying
Demonstrate domain compromise techniques

5. Key Findings

Password spraying identifies vulnerable accounts
Improper account policies increase risk
